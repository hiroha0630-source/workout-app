# 💪 筋トレ管理

体重とトレーニングだけを記録する、いちばんシンプルな筋トレ・ダイエット記録アプリ。

## 使い方（スマホ）

1. 公開URLをスマホのブラウザで開く
2. ホーム画面に追加する
   - **iPhone**: Safariの共有ボタン（□↑）→「ホーム画面に追加」
   - **Android**: Chromeのメニュー（︙）→「ホーム画面に追加」
3. ホーム画面の💪アイコンから起動（オフラインでも動作）

## 特徴

- 登録不要・広告なし・完全無料
- データは各スマホの中にのみ保存（サーバーに送信されません）
- 目標設定 / 体重記録 / トレーニング記録 / 連続日数 / CSV・レポート出力

## 注意

- データは端末ごとに別々です（家族それぞれのスマホに、それぞれの記録が保存されます）
- 機種変更のときは「設定 → バックアップを保存」でファイルを書き出し、新しい端末で「復元」してください
